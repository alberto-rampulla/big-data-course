# Esercitazioni-Big-Data

# 📊 Clinical Dataset Overview

Questo repository contiene un dataset clinico utilizzato per analisi statistiche e modelli predittivi in ambito sanitario.  
Le variabili includono informazioni demografiche, cliniche, parametri vitali, risultati di laboratorio, punteggi prognostici, funzionalità del paziente, costi e outcome.

---

## Descrizione delle Variabili

### Demografiche
| Variabile | Descrizione |
|----------|-------------|
| `age` | Età del paziente (anni) |
| `sex` | Sesso (M/F) |
| `race` | Etnia |
| `income` | Livello di reddito |
| `edu` | Anni di istruzione |

---

### Cliniche
| Variabile | Descrizione |
|----------|-------------|
| `dzgroup` | Gruppo diagnostico principale |
| `dzclass` | Classe diagnostica aggregata |
| `num.co` | Numero di comorbidità |
| `diabetes` | Presenza diabete |
| `dementia` | Presenza demenza |
| `ca` | Presenza cancro |

---

### Parametri Vitali
| Variabile | Descrizione |
|----------|-------------|
| `scoma` | Livello di coma |
| `meanbp` | Pressione arteriosa media |
| `hrt` | Frequenza cardiaca |
| `resp` | Frequenza respiratoria |
| `temp` | Temperatura corporea |
| `wblc` | Globuli bianchi |

---

### Funzionalità del Paziente

### Costi
| Variabile | Descrizione |

### Outcome
| Variabile | Descrizione |
